## Cara Menjalankan

```bash
# Install dependencies
npm install

# Run development server
cd frontend && npm run dev

# Run backend server
cd backend && npm start
```

Aplikasi akan berjalan di `http://localhost:5173` (frontend) dan `http://localhost:3000` (backend).
